# ALM Automation Kit

An enhanced Power Platform ALM Accelerator that improves automation workflows in Azure DevOps. Built on top of the CoE Starter Kit with additional capabilities.

## Features

- Automated solution export/import
- Branch strategy management
- Pull request validation pipelines
- Environment variable management
- Connection reference handling
